Цей проект містить виконання домашнього завдання з теми "Графи". 
Реалізовано створення графа транспортної мережі міст України, алгоритми обходу (DFS, BFS) та пошуку найкоротшого шляху (Дейкстри).

## Висновки
### Порівняння DFS та BFS
- **DFS (Пошук у глибину)**: Алгоритм проходить шлях від початкової вершини якомога далі вглиб перед тим, як повернутися. У нашому графі маршрут виглядав як "змійка", що заглиблюється в один напрямок.
- **BFS (Пошук у ширину)**: Алгоритм спочатку відвідує всіх найближчих сусідів поточної вершини. Це дозволило охопити всі міста, безпосередньо з'єднані з Києвом, перш ніж рухатися далі.

### Найкоротші шляхи
Алгоритм Дейкстри успішно знайшов найкоротші маршрути. Наприклад, найкоротший шлях до Львова (540 км) проходить не напряму через Житомир (140 + 400), що є оптимальним варіантом.
